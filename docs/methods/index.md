# Methods

This section is for method-level notes: preprocessing, quality control, analysis, visualization, and the reasoning behind tools or pipelines.

Use this section when the question is "what does this method do?", "which tool should I use?", or "how do these analysis pieces fit together?"

## Method Areas

| Area | Examples |
|------|----------|
| Preprocessing | fMRIPrep, FreeSurfer, MELD Graph, custom preprocessing steps |
| Quality Control | MRIQC, visual QC, checking derivative outputs |
| Analysis | GLM, RSA, ridge regression, encoding models, connectivity, ROI analysis, searchlight analysis |
| Visualization | Freeview, surface visualization, Blender brain models, figures, 3D printing |

## Difference From Workflows

Workflows are recipes. Methods are explanations and conceptual organization.

For example, "run fMRIPrep on Jubail" is a workflow. "What fMRIPrep does and how it relates to FreeSurfer, MRIQC, Docker, and Singularity" is a methods or information-style note.
